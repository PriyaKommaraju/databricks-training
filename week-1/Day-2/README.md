# Week-1 Day-2 – Aggregate Functions, GROUP BY, and HAVING

## Overview

This folder contains Day-2 SQL practice topics for Databricks training. The focus of Day-2 is understanding aggregate functions, grouping data, and filtering grouped records using the HAVING clause.

---

## Topics Covered

### 1. Aggregate Functions

Concepts

- COUNT Function
- SUM Function
- AVG Function
- MIN Function
- MAX Function
- Aggregate Calculations

Learning Areas

- Counting total records
- Finding total salary
- Calculating average values
- Identifying minimum and maximum values
- Working with aggregated data

---

### 2. GROUP BY Clause

Concepts

- Grouping Records
- Group-wise Calculations
- Department-wise Analysis
- Aggregate Functions with GROUP BY

Learning Areas

- Grouping employees by department
- Counting employees in each department
- Calculating department-wise salary totals
- Finding average salary by department
- Organizing grouped results

---

### 3. HAVING Clause

Concepts

- Filtering Grouped Data
- HAVING with Aggregate Functions
- Difference Between WHERE and HAVING
- Conditional Group Filtering

Learning Areas

- Filtering groups based on count
- Filtering groups based on average salary
- Applying conditions on grouped records
- Understanding aggregate-based filtering

---

## Learning Outcomes

After completing Day-2, you will be able to:

- Use aggregate functions efficiently
- Perform grouped analysis using GROUP BY
- Filter grouped records using HAVING
- Analyze data department-wise
- Generate summarized reports from datasets

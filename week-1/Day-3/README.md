# Week-1 Day-3 – ORDER BY and JOIN Queries

## Overview

This folder contains Day-3 SQL practice topics for Databricks training. The focus of Day-3 is understanding sorting operations using ORDER BY and combining data from multiple tables using JOIN queries.

---

# Topics Covered

## 1. ORDER BY Clause

### Concepts

- Sorting Records
- Ascending Order
- Descending Order
- Multiple Column Sorting
- Ordering Grouped Data

### Learning Areas

- Sorting employee records by salary
- Sorting records department-wise
- Ordering data in ascending and descending order
- Sorting grouped query results
- Organizing output data efficiently

---

# 2. JOIN Queries

## Concepts

- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- Matching Records Between Tables
- Combining Multiple Tables

### Learning Areas

- Retrieving employee and department details
- Combining employee and project data
- Finding records with missing matches
- Understanding relationships between tables
- Working with multiple tables in SQL

---

# Learning Outcomes

After completing Day-3, i will be able to:

- Sort records using ORDER BY
- Perform joins between tables
- Retrieve related data from multiple tables
- Understand table relationships
- Write efficient JOIN queries in SQL

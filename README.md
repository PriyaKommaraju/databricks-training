# Databricks SQL Training Repository

## Overview

This repository contains day-wise SQL practice completed as part of Databricks SQL training.  
The content includes SQL fundamentals, joins, aggregate functions, subqueries, window functions, Common Table Expressions (CTEs), and analytical queries.
